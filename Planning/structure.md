# Estructura del proyecto

La estructura general se basa en el estándar regular de una libreria de python


## Source Organization

### Root Configuration

* `setup.py` Metadata, dependencias i instrucciónes de compilación e instalación; <br>
Se define que modulos estarán en la versión final.


### Core Application

* `ASCII_Menus/` Modulos python que contendrán todas la funciónes de la librería.


### Testing

* `test/` Directorio que contendrá todos los archivos de testeo.


## Naming Conventions

* **Files**: word01_word02 (`coordinate_system.py`)
* **Directories**: Word01_Word02 (`Ascii_Menus`)
* **Directory project name**: WORD01-WORD02 (`ASCII-MENUS`)
* **Constants**: UPPER_SNAKE_CASE in dedicate file (constants.py)
